## What is Rush2.ai?

Rush2.ai is a comprehensive multi-tenant SaaS platform that enables businesses to create, manage, and scale AI-powered content generation with brand consistency and enterprise-grade security.

## Technology Stack

<Columns cols={2}>
  <Card title="Frontend" icon="window">
    - Next.js 14 with App Router
    - Tailwind CSS
    - Zustand + React Query
    - Radix UI primitives
  </Card>
  <Card title="Backend" icon="server">

    - pgvector for embeddings
    - Directus CMS
    - Temporal workflows
  </Card>
  <Card title="AI/ML" icon="robot">

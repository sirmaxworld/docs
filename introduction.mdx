## What is Rush2.ai?

Rush2.ai is a comprehensive multi-tenant SaaS platform that enables businesses to create, manage, and scale AI-powered content generation with brand consistency and enterprise-grade security.

## Technology Stack

### Frontend
- **Framework**: Next.js 14 with App Router
- **Styling**: Tailwind CSS with custom design system
- **State Management**: Zustand + React Query
- **UI Components**: Radix UI primitives

### Backend
- **Database**: PostgreSQL with Prisma ORM
- **Vector Storage**: pgvector for embeddings
- **CMS**: Directus for content management
- **Workflow**: Temporal for orchestration
- **Cache**: Redis for rate limiting

### AI/ML
- **LLM Providers**: OpenAI, Anthropic, Perplexity
- **Image Generation**: Recraft, Stability AI
- **Video Processing**: Shotstack
- **Embeddings**: OpenAI Ada-002

### Infrastructure
- **Hosting**: Vercel Edge Functions
- **Storage**: Cloudflare R2
- **CDN**: Cloudflare
- **Monitoring**: Sentry, PostHog

## Getting Started

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/sirmaxworld/rush2ai_v3.git
    cd rush2ai_v3
    ```
  </Step>
  <Step title="Install Dependencies">
    ```bash
    npm install
    ```
  </Step>
  <Step title="Configure Environment">
    ```bash
    cp .env.example .env.local
    # Edit .env.local with your API keys
    ```
  </Step>
  <Step title="Run Development Server">
    ```bash
    npm run dev
    ```
  </Step>
</Steps>
